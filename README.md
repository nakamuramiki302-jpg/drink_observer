# 飲み過ぎalert 🍺

あなたの飲酒を見守るWebアプリケーション

## 概要

「飲み過ぎalert」は、アルコール摂取量を記録し、血中アルコール濃度（BAC）をリアルタイムで可視化するWebアプリです。
Widmark式を用いた科学的な計算により、現在の酔いレベルを正確に把握できます。

## 主な機能

- ✨ **本日のアルコール摂取量表示** - 純アルコール換算でmlで表示
- 🎯 **酔いレベルメーター** - 血中アルコール濃度を円形グラフで可視化
- 🍷 **飲み物記録** - プリセット（ビール、ワイン、日本酒など）+ 手入力対応
- 📊 **タイムライン** - いつ何を飲んだかの記録
- 📈 **BAC推移グラフ** - 時間経過による血中アルコール濃度の変化を可視化
- ⚙️ **体格情報設定** - 体重と性別を考慮した正確な計算
- 💾 **データ永続化** - ブラウザのlocalStorageに自動保存

## 酔いレベルの基準

- 😊 **シラフ** (0-0.02%)
- 😄 **爽快期** (0.02-0.04%) - 陽気になる
- 😜 **ほろ酔い期** (0.04-0.1%) - 気分が良くなる
- 🥴 **酩酊初期** (0.1-0.15%) - 千鳥足、呂律が回らない
- 👿 **酩酊期** (0.15-0.3%) - まともに立てない
- 💀 **泥酔期** (0.3%以上) - 意識喪失の危険

## 使い方

1. ブラウザで `index.html` を開く
2. 初回起動時に体重と性別を入力
3. 飲んだお酒をプリセットまたは手入力で記録
4. リアルタイムで酔いレベルとグラフが更新される

## 技術スタック

- React 18 (CDN)
- Chart.js - グラフ描画
- Tailwind CSS - スタイリング
- localStorage - データ永続化

## 計算方法

血中アルコール濃度はWidmark式を使用して計算しています：

```
BAC = (A / (W × r)) - (β × t)

A: 摂取した純アルコール量（g）
W: 体重（kg）
r: 分布容積（男性: 0.68, 女性: 0.55）
β: アルコール代謝速度（0.015% / 時間）
t: 経過時間（時間）
```

## 注意事項

- 血中アルコール濃度は目安です。実際の値とは異なる場合があります
- 飲酒運転は絶対にしないでください
- 適度な飲酒を心がけましょう
